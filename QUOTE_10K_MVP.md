# QUOTATION

**Project:** Prototype Implementation and Evaluation Support for Permissioned Blockchain System

## Client Request

Development of a minimum viable prototype based on the provided specification for a permissioned blockchain solution using Hyperledger Fabric, with emphasis on demonstrating implementation and proof of governance and security rules.

## Scope of Work

The quoted amount covers delivery of a focused academic MVP prototype aligned to the core objectives of the specification.

Included in scope:

- requirements analysis and translation of the specification into an MVP implementation plan
- setup of a local Hyperledger Fabric prototype environment using Docker
- implementation of core blockchain business logic for the selected shipment workflow
- configuration of governance and endorsement rules required for the demonstration
- implementation of confidentiality logic using restricted/private data handling
- implementation of off-chain document hash anchoring for compliance verification
- simple controlled interaction layer for transaction submission and query
- positive and negative test scenarios to demonstrate the enforcement rules
- evidence support for academic demonstration, including screenshots and outputs
- concise technical documentation explaining the prototype structure and operation
- limited revision support necessary to complete the MVP

## MVP Outcomes

The MVP is intended to demonstrate:

- governance control through endorsement rules
- confidentiality through restricted/private data visibility
- bounded transparency through audit traceability
- compliance through off-chain document verification by on-chain hash reference
- controlled interaction with the blockchain through a managed access layer

## Exclusions

This quotation is for an academic MVP prototype and does not include:

- production deployment
- cloud hosting
- advanced frontend or dashboard development
- enterprise-grade security hardening
- unlimited revisions
- full-scale platform development beyond the agreed MVP scope
