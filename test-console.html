5 · Hybrid Compliance — Document Hash Anchoring
Upload document POST /documents/upload
Stores file off-chain; anchors SHA-256 hash on-chain via RecordDocumentHash chaincode.
Will be base64-encoded and sent as document body.
Verify document integrity Expected matches: true
Re-hashes stored file and compares with on-chain record.
Simulate tampering + re-verify Expected matches: false
Overwrites the stored file, then re-verifies to prove hash mismatch is detected.
Requires docker exec access from the terminal:
docker exec api-gateway sh -c 'echo "TAMPERED" > /storage/documents/DOC100'
